JavaScript Basics: let, const, and Operators

1. let

let is used to declare a variable that you might want to change later. It only exists inside the block of code (enclosed by curly braces {}) where you create it.

Example:

let favoriteColor = "blue";
favoriteColor = "green"; // You can change it to "green"

2. const

const is used to declare a variable that you don't want to change. It also only exists inside the block of code where you create it. Once a value is assigned, it cannot be reassigned, but you can still modify the contents if it’s an object or an array.

Example:

const birthYear = 2004;
// birthYear = 2005; // This would cause an error, because you can’t change it

const fruits = ["Apple", "Banana"];
fruits.push("Orange"); // You can still add items to the list

3. Operators

Operators are symbols that let you do things with variables, like math or comparisons.

Arithmetic Operators

These are for doing math:

  • + : Adds things together.
  • - : Subtracts one thing from another.
  • * : Multiplies things.
  • / : Divides one thing by another.
  • % : Gives you the remainder after dividing two numbers.

Example:

let total = 10 + 5; // 15
let remainder = 10 % 3; // 1 (because 10 divided by 3 leaves a remainder of 1)

Assignment Operators

These are for putting values into variables:

  • = : Puts a value into a variable.
  • += : Adds something to what’s already in the variable.
  • -= : Subtracts something from what’s already in the variable.

Example:

let score = 0;
score += 10; // Now score is 10 (it added 10 to the original 0)

Comparison Operators

These are for comparing things to see if they’re equal, greater, less, etc.:

  • == : Checks if the values are the same (ignores type).
  • === : Checks if the values and types are the same (more strict).
  • != : Checks if values are different.
  • !== : Checks if values or types are different.
  • > : Greater than.
  • < : Less than.

Example:

console.log(5 == "5"); // true (compares values only)
console.log(5 === "5"); // false (compares values and types)

Logical Operators

These are for combining or reversing conditions:

  • && : AND (both things must be true).
  • || : OR (at least one thing must be true).
  • ! : NOT (reverses the condition).

Example:

let isSunny = true;
let isWeekend = false;

console.log(isSunny && isWeekend); // false (both need to be true)
console.log(isSunny || isWeekend); // true (at least one is true)
console.log(!isSunny); // false (because it was true, and `!` makes it false)
